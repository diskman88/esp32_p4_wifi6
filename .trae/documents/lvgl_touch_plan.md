# LVGL 添加触摸功能实现计划

## 1. 当前状态分析

### 1.1 触摸设备配置
从 `board_devices.yaml` 中可以看到：
- **设备名称**: `lcd_touch`
- **芯片**: GT911
- **类型**: `lcd_touch_i2c`
- **分辨率**: 1024x600（与 LCD 相同）
- **镜像配置**: mirror_x: true, mirror_y: true

### 1.2 现有代码状态
- LVGL 显示已正常工作
- 背光已初始化并设置亮度
- 缺少触摸输入支持

---

## 2. 实现计划

### 2.1 修改文件列表

| 文件路径 | 修改类型 | 说明 |
|---------|---------|------|
| `main/mpython_idf.c` | 修改 | 添加触摸初始化和 LVGL 触摸输入 |

### 2.2 步骤详情

#### 步骤 1: 添加触摸设备初始化
- 使用 `esp_board_manager_init_device_by_name("lcd_touch")` 初始化触摸设备
- 获取触摸设备句柄

#### 步骤 2: 添加 LVGL 触摸输入端口
- 使用 `lvgl_port_add_touch()` 添加触摸输入
- 配置触摸坐标转换（考虑镜像配置）

#### 步骤 3: 测试触摸交互
- 确保按钮点击和滑块拖动能正常响应触摸输入

---

## 3. 技术要点

### 3.1 GT911 触摸控制器
- 通过 I2C 通信
- 支持多点触摸
- 需要配置镜像参数以匹配 LCD 显示

### 3.2 LVGL 触摸集成
- 使用 `esp_lvgl_port` 组件提供的触摸接口
- 确保触摸坐标与显示坐标正确映射

---

## 4. 预期输出

完成后，LVGL UI 将支持触摸交互：
- 按钮可通过触摸点击
- 滑块可通过触摸拖动
- 触摸坐标准确映射到屏幕位置
